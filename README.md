# 富邦產險_風險評估系統 RiskMeasure

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 7.3.9.

## 功能需求

1. 使用者可以上傳圖片當底圖
1. 使用者可以在圖片上面畫線、框、箭頭、文字
1. 可以分圖層（底圖、使用者作畫的內容）顯示
1. 編輯完的內容另存成圖片
